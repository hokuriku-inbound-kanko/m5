# m5

北陸地方における観光客動線分析のためのWi-Fiパケットセンシングによるオープンデータ。金沢大学が収集。

## 背景
このデータセットは、Wi-Fiパケットセンシング技術を活用し、6つの指定エリア（area1–area6）における観光客の行動や移動パターンを分析したものです。時間ごとの訪問傾向を追跡し、観光計画やインフラの最適化を支援します。方法論の詳細は、以下の参考論文をご覧ください。

## データファイル
リポジトリには以下のファイルが含まれます：
- **CSVデータ**:
  `kaiteki/area1_m5.csv` から `kaiteki/area6_m5.csv`
  これらのファイルは、タイムスタンプ付きのWi-Fiアクティビティのカウントを記録しています。`00_M5_facility list.csv` を使用して、area1–area6のエリア番号を現実世界の場所にマッピングしてください。
- **Webビューアー**:
  `kaiteki/index.html` は、Chart.jsを使用してデータのインタラクティブな可視化を提供します。

## データ列
CSVファイルには以下の列が含まれます：
- `year`, `month`, `day`, `hour`, `minute`, `second`: 測定のタイムスタンプ。
- `count`: 指定された時間における検出されたWi-Fiパケットまたはデバイスの数。

## Webビューアー
組み込みのWebダッシュボード（`kaiteki/index.html`）により、データを視覚的に探索できます。[Tailwind CSS](https://tailwindcss.com/) と [Chart.js](https://www.chartjs.org/) を使用して、レスポンシブなブラウザベースの分析を実現しています。
